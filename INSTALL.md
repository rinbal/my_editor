# Install MyEditor

One file, one click. Download the version for your system from the
[latest release](https://github.com/rinbal/my_editor/releases/latest), then open it.

## Windows

Download `my-editor-x.y.z-windows-setup.exe` and run it. If Windows shows
"Windows protected your PC", click **More info**, then **Run anyway**. Launch it
from the Start Menu.

## macOS

Download the `.dmg` (`-arm64` for Apple Silicon, `-intel` for older Intel Macs),
open it, and drag the app into **Applications**. On the first launch, right-click
the app and choose **Open**, then **Open** again.

## Linux

Download `my-editor-x.y.z-linux-x86_64.AppImage`, make it executable
(`chmod +x my-editor-*.AppImage`, or Properties > Allow executing), then
double-click it.

---

The app is unsigned, so the first launch shows a one-time security prompt on
every system. After that it opens with a normal double-click. Need more detail
or troubleshooting? See [DOWNLOAD.md](DOWNLOAD.md).
